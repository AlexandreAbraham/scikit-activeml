API Reference¶

This is an overview of the API.

skactiveml.base: Base classes¶

The skactiveml.base package implements the base classes for skactiveml.

QueryStrategy

Base class for all query strategies in scikit-activeml.

SingleAnnotPoolBasedQueryStrategy

Base class for all pool-based active learning query strategies with a single annotator in scikit-activeml.

MultiAnnotPoolBasedQueryStrategy

Base class for all pool-based active learning query strategies with multiple annotators in scikit-activeml.

SkactivemlClassifier

Base class for scikit-activeml classifiers such that missing labels, user-defined classes, cost-sensitive classification (i.e., cost matrix), and multiple labels per sample can be handled.

ClassFrequencyEstimator

Extends scikit-activeml classifiers to estimators that are able to estimate class frequencies for given samples (by calling 'predict_freq').

AnnotModelMixin

Base class of all annotator models estimating the performances of annotators for given samples.

skactiveml.pool: Pool-based strategies¶

The skactiveml.pool package implements query strategies for pool-based active learning.

Classes¶

RandomSampler

Random sampling.

McPAL

Multi-class Probabilistic Active Learning

UncertaintySampling

Uncertainty Sampling

EpistemicUncertainty

Epistemic uncertainty sampling query strategy for two class problems.

ExpectedErrorReduction

Expected Error Reduction

QBC

The Query-By-Committee (QBC) algorithm minimizes the version space, which is the set of hypotheses that are consistent with the current labeled training data.

FourDS

Implementation of the pool-based query strategy 4DS for training a CMM [1].

ALCE

Active Learning with Cost Embedding (ALCE)

skactiveml.pool.multi: Multi Annotator Pool-based strategies¶

Classes¶

IEThresh

The strategy 'Interval Estimation Threshold' (IEThresh) [1] is useful for addressing the exploration vs.

IEAnnotModel

This annotator model relies on 'Interval Estimation Learning' (IELearning) for estimating the annotation performances, i.e., labeling accuracies, of multiple annotators [1].

MultiAnnotWrapper

Implementation of a wrapper class for scikit-learn pool-based active learning query strategies with a single annotator such that it transforms the query strategy for the single annotator into a query strategy for multiple annotators by randomly choosing an annotator and setting the labeled matrix to a labeled vector by an aggregation function.
