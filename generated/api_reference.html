This annotator model relies on 'Interval Estimation Learning' (IELearning) for estimating the annotation performances, i.e., labeling accuracies, of multiple annotators [1].

Implementation of a wrapper class for scikit-learn pool-based active learning query strategies with a single annotator such that it transforms the query strategy for the single annotator into a query strategy for multiple annotators by randomly choosing an annotator and setting the labeled matrix to a labeled vector by an aggregation function.
